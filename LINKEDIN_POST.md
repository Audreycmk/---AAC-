🗣️ **Bridging Communication Gaps: An AAC Solution for Hong Kong's Elderly** 🇭🇰

I'm excited to share my latest project - an Augmentative and Alternative Communication (AAC) web application designed specifically for Hong Kong's aging population.

**The Challenge:**
Originally developed for stroke survivors who have lost their ability to speak, this project addresses a critical need in Hong Kong's healthcare landscape. But during development, I realized it could solve an even broader problem: helping elderly individuals living alone communicate effectively with their foreign domestic helpers.

**The Solution:**
This bilingual AAC app features:
✅ Large, senior-friendly buttons with intuitive icons
✅ Instant Cantonese voice synthesis
✅ English subtitles on every phrase
✅ English pronunciation support for cross-cultural communication
✅ 50+ pre-programmed phrases covering daily needs, medical emergencies, and emotional expressions
✅ Calculator-style number input for quantities and measurements
✅ Custom sentence builder for personalized communication

**Real-World Impact:**
Many elderly Hong Kong residents rely on foreign domestic helpers for daily care. Language barriers can create dangerous situations, especially during medical emergencies. This app empowers them to communicate essential needs clearly - from "I need my medicine" to "I'm feeling unwell" - in both Cantonese and English.

**Coming Soon:**
🇮🇩 Indonesian language support to facilitate even better communication with the largest domestic helper demographic in Hong Kong.

**Tech Stack:**
Built with Next.js 15, TypeScript, and Web Speech API - ensuring offline capability and accessibility across all devices.

This project reminds me that technology's greatest value lies in solving real human problems. When design meets empathy, we can create tools that truly make a difference in people's daily lives.
