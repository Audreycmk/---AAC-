# Audrey Chung 輔助通訊 (AAC)

一個專為香港康復者設計的輔助和替代性溝通（AAC）Web 應用程式，使用 Next.js 和 Web Speech API 構建，支援廣東話語音輸出。

## 核心特點

✨ **快、準、大** - 三大設計原則
- 🔘 **大按鈕** - 易於點擊的大型按鈕設計（最小 180px 高度）
- ⚡ **快反應** - 即時語音播放，無延遲
- 🎯 **準語音** - 使用瀏覽器內建 Web Speech API，支援廣東話 (zh-HK)

## 主要功能

### 🗣️ 語音功能
- 廣東話語音合成（Text-to-Speech）
- 可調整語速（0.5x - 2.0x）
- 可調整音量（0% - 100%）
- 自動停止前一個語音播放

### 📝 輸入方式
- **預設短語** - 34 個常用廣東話短語，一鍵播放
- **自訂輸入** - 輸入任何文字進行語音播放
- **句子啟動器** - 快速開始常用句型（我想、可唔可以、幫我、我要）
- **智能建議** - 根據句子啟動器顯示相關詞語建議（共 40 個建議詞）

### 📋 分類系統
- **日常** - 你好、多謝、唔該、對唔住、再見、好/唔好、係/唔係、明白/唔明白
- **飲食** - 飲水、肚餓、食飯、飲奶、水果（蘋果、橙、香蕉）
- **醫療** - 唔舒服、頭痛、睇醫生、食藥、量體溫、去廁所
- **情緒** - 開心、傷心、攰、擔心、熱、凍
- **求助** - 幫幫我、休息、需要幫忙、等等

### 📜 其他功能
- **歷史記錄** - 自動保存最近 10 條語音記錄，可快速重複使用
- **響應式設計** - 支援手機、平板和電腦
- **美觀界面** - 配色主題：深藍色 (#1e3a5f) + 橙色 (#f97316) + 米色 (#f5f5dc)
- **動畫效果** - 按鈕懸停、縮放、旋轉等流暢動畫
- **無障礙優化** - ARIA 標籤、鍵盤支援
- **離線存儲** - 使用 localStorage 保存歷史記錄

## 技術堆疊

- **Next.js 15** - React 框架（App Router）
- **TypeScript** - 類型安全
- **Tailwind CSS** - 現代化樣式
- **Web Speech API** - 瀏覽器內建語音合成

## 開始使用

### 安裝依賴

```bash
npm install
```

### 啟動開發伺服器

```bash
npm run dev
```

開啟瀏覽器訪問 [http://localhost:3000](http://localhost:3000)

### 構建生產版本

```bash
npm run build
npm start
```

## 瀏覽器支援

為了獲得最佳體驗，建議使用以下瀏覽器：

- ✅ Google Chrome / Edge（推薦）
- ✅ Safari
- ⚠️ Firefox（部分支援廣東話）

## 使用方法

1. 開啟應用程式
2. 點擊任何短語按鈕
3. 系統會立即播放廣東話語音
4. 使用頂部的分類按鈕快速篩選

## 自訂短語

要新增或修改短語，編輯 `app/components/AACApp.tsx` 檔案中的 `PHRASES` 陣列：

```typescript
const PHRASES = [
  { id: 1, text: '你好', category: '問候' },
  // 在這裡新增更多短語
];
```
