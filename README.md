# Audrey Chung 輔助通訊 (AAC)

一個專為香港康復者設計的輔助和替代性溝通（AAC）Web 應用程式，使用 Next.js 和 Web Speech API 構建，支援廣東話語音輸出。

## 核心特點

✨ **快、準、大** - 三大設計原則
- 🔘 **大按鈕** - 易於點擊的大型按鈕設計（最小 180px 高度）
- ⚡ **快反應** - 即時語音播放，無延遲
- 🎯 **準語音** - 使用瀏覽器內建 Web Speech API，支援廣東話 (zh-HK)

## 主要功能

### 🗣️ 語音功能
- 廣東話語音合成（Text-to-Speech）
- 支援多種語言（廣東話、美式英語、澳洲英語）
- 可選擇不同聲音
- 可調整語速（0.5x - 2.0x）
- 可調整音量（0% - 100%）
- 按鈕點擊即時語音播放（速度 1.0x）
- 自動停止前一個語音播放

### 📝 輸入方式
- **預設短語** - 50+ 個常用廣東話短語，一鍵播放
- **自訂輸入** - 輸入任何文字進行語音播放
- **句子啟動器** - 快速開始常用句型（我想、我要、幫我、唔記得等）
- **智能建議** - 根據句子啟動器顯示相關詞語建議（50+ 個建議詞）
- **計算器界面** - 量詞分類使用計算器式數字鍵盤輸入

### 📋 分類系統
- **個人物品** - 銀包、手機、眼鏡、口罩、鎖匙、袋、外套、橡筋
- **家居用品** - 電視、門、冷氣、暖爐、窗、燈、匙羹、叉
- **水果** - 提子、香蕉、蘋果、橙、火龍果、菠蘿、桃、榴槤
- **地方** - 廁所、銀行、超市、李鄭屋中心、醫院、酒樓、公園、茶餐廳
- **量詞** - 數字輸入（0-9）+ 時間單位（日、月、年、歲、分鐘、小時、禮拜）+ 度量單位（斤、両、寸、磅、厘米）

### 🔢 計算器功能
- **數字鍵盤** - 0-9 數字按鈕，排列為 7-9、4-6、1-3
- **小數點** - 支援輸入小數
- **清除按鈕** - 一鍵清除輸入
- **時間單位** - 日、月、年、歲、分鐘、小時、禮拜
- **度量單位** - 斤、両、寸、磅、厘米
- **點擊播放** - 每個按鈕點擊時即時語音播放
- **顯示屏** - 可點擊播放當前輸入內容
- **響應式布局** - 桌面版左右分欄，手機版上下排列

### 👤 用戶管理系統
- **登入碼登入** - 快速登入（如 btc2026）
- **管理員登入** - 郵箱密碼登入
- **試用管理** - 14 天試用期、無限制、無試用
- **自訂圖像** - 支援用戶專屬圖像（如 btc2026 用戶使用自訂照片）
- **用戶儀表板** - 管理員可管理所有用戶

### 📜 其他功能
- **歷史記錄** - 自動保存最近 10 條語音記錄，可快速重複使用
- **響應式設計** - 支援手機、平板和電腦，自適應布局
- **美觀界面** - 配色主題：深藍色 (#1e3a5f) + 橙色 (#f97316) + 米色 (#f5f5dc)
- **動畫效果** - 按鈕懸停、縮放、按壓等流暢動畫
- **無障礙優化** - ARIA 標籤、鍵盤支援
- **離線存儲** - 使用 localStorage 保存歷史記錄和用戶設定
- **客人模式** - 未登入用戶可使用數字鍵盤，其他功能需登入

## 技術堆疊

## 使用方法

### 基本使用
1. 開啟應用程式
2. 選擇分類（個人物品、家居用品、水果、地方、量詞）
3. 點擊任何短語按鈕，系統會立即播放廣東話語音
4. 使用側邊選單快速切換分類

### 量詞（數字）輸入
1. 選擇「量詞」分類
2. 使用計算器式鍵盤輸入數字（0-9、小數點）
3. 點擊時間單位（日、月、年、歲、分鐘、小時、禮拜）
4. 點擊度量單位（斤、両、寸、磅、厘米）
5. 點擊顯示屏或播放按鈕播放完整內容

### 自訂句子
1. 點擊「自訂句子」按鈕
2. 選擇句子啟動器（如「我想」）
3. 選擇建議詞語或從詞彙表選擇
4. 點擊播放按鈕

### 用戶登入
- **一般用戶**：點擊登入按鈕，輸入登入碼（如 btc2026）
- **管理員**：點擊「Admin」，使用郵箱和密碼登入
- **客人模式**：無需登入即可使用數字鍵盤

## 自訂短語

要新增或修改短語，編輯 `app/components/AACApp.tsx` 檔案中的 `PHRASES` 陣列：

```typescript
const PHRASES = [
  { id: 1, text: '你好', en: 'Hello', category: '日常', icon: '👋' },
  // 在這裡新增更多短語
];
```

### 添加用戶專屬圖像

1. 將圖片放入 `/public` 目錄（支援 .jpg、.png）
2. 編輯 `app/components/PhrasesGrid.tsx`
3. 在 `BTC2026_IMAGE_MAP` 中添加映射：

```typescript
const BTC2026_IMAGE_MAP: Record<number, string> = {
  35: '/wallet.jpg',  // 短語 ID: 圖片路徑
  // 添加更多映射
};
```
